# 🇹🇼 台灣統一發票對獎與自動查詢器

自動同步財政部最新中獎號碼，提供快速對獎、末三碼快速篩選、中獎音效、離線紀錄與中獎發票管理功能。

## 🌟 核心特色
1. **資料自動化更新**：透過 GitHub Actions 每天定時爬取財政部官網最新開獎資訊，完全不需手動更新。
2. **末三碼快速篩選**：仿照手動對發票的習慣，輸入發票末三碼如果與中獎號碼不符，立即判斷「未中獎」，節省大量對獎時間；若相符則播放提示音引導輸入完整號碼。
3. **8位數精確對獎**：輸入完整發票號碼，自動比對特別獎、特獎、頭獎及各尾數二~六獎，算出最終獲得之獎金。
4. **離線與紀錄功能**：自動儲存已對獎歷史明細，統計累計對發票張數與中獎總額（數據儲存於瀏覽器本地，完全保障隱私）。
5. **精美科技感 UI**：暗黑色系玻璃擬物風格與霓虹高亮提示，並內建中獎撒花特效與網頁音效。

## 🛠️ 技術堆疊
* **前端核心**：Vanilla HTML5, CSS3 (科技風玻璃擬物/網格排版), JavaScript
* **自動爬蟲**：Python 3 (urllib + re 結構化解析)
* **自動化維護**：GitHub Actions 定時自動執行爬網並發布網頁 (GitHub Pages)

## 📁 專案結構說明
* `index.html`: 快速對獎器主頁面結構
* `style.css`: 科技風玻璃半透面板與動態特效樣式
* `app.js`: 音效合成、撒花粒子與對獎比對主邏輯
* `cron_crawler.py`: 爬取財政部中獎號碼的 Python 腳本
* `.github/workflows/auto_crawl.yml`: 定時自動化爬蟲 GitHub Actions 工作流
* `data/winning_numbers.json`: 當前最新中獎號碼數據庫（由爬蟲自動更新）
